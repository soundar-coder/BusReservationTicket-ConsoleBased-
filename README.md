Bus Reservation System (Console-Based)

A Java Console Application that simulates a real-world Bus Reservation System.
Built using Core Java, OOP, Collections, JDBC, and Multithreading for handling seat booking concurrency.

Features

User Registration & Login 

Search Buses by route

View Available Seats

Book / Cancel Tickets

Passenger Details Management

Payment Simulation

Multithreading

Prevents double-booking of the same seat

Ensures synchronized seat allocation

JDBC Connectivity (MySQL )

Exception Handling
